# Claude Personal Finance Dashboard

## Overview

Claude Personal Finance Dashboard is a Python command-line tool that helps users track income, expenses, savings, and spending patterns.

This project demonstrates data analysis and automation by allowing users to enter transactions, automatically categorize expenses, generate financial reports, and identify savings opportunities.

## Features

* Tracks income and expense transactions
* Automatically categorizes expenses using keyword matching
* Saves transactions to a CSV file
* Calculates total income
* Calculates total expenses
* Calculates net savings
* Calculates savings rate
* Shows spending by category
* Identifies the highest spending category
* Highlights savings opportunities
* Runs locally with no API key required

## Technologies Used

* Python
* CSV File Storage
* Data Analysis
* Rule-Based Automation
* Command Line Interface (CLI)

## How to Run

1. Download the project files.

2. Open a terminal in the project folder.

3. Run:

```bash
python personal_finance_dashboard.py
```

No API key, paid credits, or advanced setup is required.

## Example Use Case

A user enters income from work and expenses such as rent, groceries, transportation, entertainment, and shopping.

The dashboard can help the user:

* Track where their money is going
* See total income and expenses
* Calculate monthly savings
* Identify their highest spending category
* Find areas where they may be able to save more money
